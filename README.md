> Una aplicación web simple, dinámica y visual para gestionar y planificar el progreso académico a través de una malla curricular interactiva, validando automáticamente los prerrequisitos.

**Caso de estudio actual:** Malla de la carrera de **Ingeniería Civil en Computación e Informática** (Plan 2018) de la **Universidad Mayor**.

---

## 🏗️ Estructura Tecnológica

El proyecto está construido utilizando tecnologías web estándar (Vanilla), sin dependencias externas complejas:

*   **📄 HTML Básico:** Estructura semántica de la interfaz.
*   **🎨 CSS Puro:** Diseño y estilos creados desde cero, sin frameworks.
*   **⚡ JavaScript Modular:** Lógica de la aplicación e interactividad.
*   **📊 JSON:** Fuente de datos estructurada para los ramos y sus requisitos.

---

## 🚀 ¿Cómo usarlo?

Probar el proyecto en tu computadora es muy rápido:

1. **Clona o descarga** este proyecto en tu equipo local.
2. Abre la carpeta del proyecto y utiliza **Live Server** (en VS Code) o cualquier otro servidor web local.
3. **¡Listo!** Comienza a navegar y marcar tus ramos aprobados en la malla.

---

## 💡 Nota Importante

> ℹ️ **Demo Educativa:** Esta aplicación no utiliza frameworks frontend (como React o Angular) ni bases de datos. Está pensada y diseñada principalmente como un recurso didáctico para aprendizaje.
